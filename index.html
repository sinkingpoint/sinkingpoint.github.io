QInit

QInit

I decided to learn Rust, so started the same way I learn all low level languages - I built a Linux Distribution with it. QInit is a distribution very minimal external code - everything from the init scripts to the shell to the coreutils are all written by me. I'm most proud of the Init Daemon that controls service scheduling, and the IPC daemon "Freud" that allows encrypted and trusted message passing in a one to many model between processes
