Alertmanager Bouncer

Alertmanager Bouncer

Alertmanager Bouncer is a business logic proxy that sits in front of a Prometheus Alertmanager and applies logic to block or proxy requests to the Alertmanager. This allows enforcement of rules like silences having authors attached, and ratelimiting services sending alerts. It is currently running in production in a number of large scale environments.

Kiora

Kiora is a drop in replacement for Prometheus Alertmanager, managing alerting lifecycles while providing a much more flexible and transparent way to manage alerts.
